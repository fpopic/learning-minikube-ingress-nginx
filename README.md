# learning-minikube-ingress-nginx

## How to run

Environment
- m1 macOs
- minikube version: v1.27.0

Run [./minikube-up.sh](./minikube-up.sh) to spin up and [./minikube-down.sh](./minikube-down.sh) to stop everything.
